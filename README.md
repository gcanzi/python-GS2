# Orbit Sentinel 🛰️🌍
**Sistema Inteligente de Monitoramento Climático Inspirado em Tecnologias Orbitais**

Projeto desenvolvido para a disciplina de *Computational Thinking With Python* (1º Ano de Engenharia de Software - FIAP). O sistema atua na prevenção de desastres naturais processando dados de sensores e emitindo alertas baseados em regras de negócio.

## 🛠️ Tecnologias Utilizadas
- **Python 3.10+**
- **JSON** para persistência local de dados.
- Estruturação modular aplicando princípios de *Clean Code*.

## ⚙️ Funcionalidades
1. **Cadastro e Validação:** Registro de sensores ambientais.
2. **Monitoramento e Alertas:** Registro de medições com análise inteligente em tempo real (calor extremo, baixa umidade, etc.).
3. **Persistência de Dados:** Salvamento automático utilizando arquivos JSON.
4. **Tratamento de Exceções:** Sistema robusto contra entradas incorretas e erros de I/O.
5. **Relatórios e Estatísticas:** Agregação de dados para apoio à tomada de decisão.

## 🚀 Como Executar

1. Clone o repositório para sua máquina.
2. Abra o terminal na pasta raiz do projeto.
3. Execute o comando:
   ```bash
   python main.py
